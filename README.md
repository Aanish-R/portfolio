# Personal Portfolio - Anish Reji

A premium, creative portfolio website built with HTML, CSS, and JavaScript.

## Features
- Modern, responsive design with Glassmorphism.
- Dark/Light mode support.
- Custom scroll animations and cursor glow effects.
- Branding icons for professional tools.

## Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript.
- **Icons**: Lucide Icons, DevIcons.
- **Fonts**: Google Fonts (Outfit, Inter).

## Projects Highlighted
- **KTU Result Analyzer**: A data analytics tool for academic performance tracking.
