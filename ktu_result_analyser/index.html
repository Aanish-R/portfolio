KTU Result Analyzer

A powerful academic tool designed to parse, analyze, and visualize result data from Kerala Technological University (KTU). Transform unstructured PDF results into actionable insights.

View Source Code
Capabilities

Key Features

PDF Parsing

Automatically extracts student marks, names, and register numbers from official KTU result PDFs with high accuracy.

Visual Analytics

Generates interactive charts showing pass percentages, GPA distribution, and subject-wise performance.

Excel Export

Converts complex result data into clean, formatted Excel sheets for easy record-keeping and offline analysis.

Smart Filtering

Filter results by department, year, or specific subjects to identify trends and areas for improvement.

How to Run Locally

1. Clone requirements

git clone https://github.com/Aanish-R/portfolio.git
cd portfolio/ktu_result_analyser

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

The application will start at http://127.0.0.1:5000
